# Microsoft Learn 📖

## Copilot 拡張性公式ドキュメント

Copilot を拡張する Microsoft Learn コンテンツを、すべて 1 か所で確認できます。**Official Collection: Microsoft 365 Copilot extensibility** には、[https://aka.ms/extend-microsoft365-copilot](https://aka.ms/extend-microsoft365-copilot) からアクセスしてください。

## MCP サーバー

GitHub Copilot などのコーディング エージェントを使用して Microsoft Learn コンテンツでエクスペリエンスを強化している場合は、MCP サーバーをクライアントに接続しましょう。

エンドポイント `https://learn.microsoft.com/api/mcp` を手動で設定するか、[MCP Servers for agent mode](https://code.visualstudio.com/mcp) にアクセスし、 **Microsoft Docs** ボタンをクリックするだけで Visual Studio Code にインストールできます。
