# ラボ MAB - Copilot Studio Lite によるエージェント作成

**Copilot Studio Lite** エージェントは、Microsoft 365 Copilot と同じインフラストラクチャ、オーケストレーター、基盤モデル、セキュリティ制御を活用して宣言型のエージェントを作成する最も簡単な方法です。これにより、一貫性のある親しみやすいユーザー エクスペリエンスが保証されます。Copilot Studio Lite エージェントでは、カスタム ナレッジ ベースやカスタム機能/スキルも構成できます。

![Copilot Studio Lite エージェントのアーキテクチャ図。最下層には Microsoft 365 Copilot の基盤モデルと同じオーケストレーターがあります。エージェントはカスタム ナレッジおよびグラウンディング データ、さらにカスタム機能/スキルを提供します。ユーザー エクスペリエンスは Microsoft 365 Copilot Chat に完全統合されています。](../../../assets/images/copilot-studio-lite-agent.png)

これは、Microsoft 365 Copilot 用のパーソナル エージェントを作成する最も簡単な方法です。Microsoft 365 Copilot Chat と Microsoft 365 Copilot のネイティブ UI に統合されています。Microsoft が提供するテンプレート セットから始めることも、ゼロから開始することもできます。自然言語で作成したいエージェントの目的を説明するだけで、Copilot Studio Lite が意図を理解し、プロンプトに基づいてエージェントを作成します。または、設定ページからエージェントを構成することも可能です。既定ではエージェントは個人利用のみですが、必要に応じて組織内の特定のユーザー、あるいは組織内のすべてのユーザーと共有できます。

![Copilot Studio Lite のユーザー インターフェイス。ユーザーに対して、エージェント テンプレートを選択するか、自然言語でエージェントの目的を記述するかを求めるダイアログが表示されています。](../../../assets/images/make-global-intro/copilot-agent-builder-01.png)

Copilot Studio Lite で作成したエージェントは、次のようなナレッジ ベースを利用できます。

- 一般的な Web コンテンツ
- SharePoint Online に保存されているドキュメントのコンテンツ
- Microsoft Copilot Connectors

また、オンザフライで画像を生成したり、自然言語をコードに変換したりすることもできます。

エージェントは Microsoft 365 Copilot Chat で利用でき、Microsoft 365 Copilot のライセンスを持つユーザーであれば誰でも使用できます。  
対象となるナレッジ ベースやユーザーのライセンス形態によっては、従量課金サービスの料金が発生する場合があります。
