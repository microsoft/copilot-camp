# Microsoft Agent 365

Agent 365 is the control plane for agents—giving your IT team one place to observe, govern, and secure every agent across your organization as you move from agentic AI experimentation to enterprise-scale operations.

## Stay in control as agents become part of everyday work

Agent 365 enables you to monitor and manage agents in real time, protect all agents end-to-end, and establish guardrails for agents and users. Key capabilities include:

- **Registry** - Get a complete view of all agents in your organization, including agents built with Microsoft AI platforms, agents from ecosystem partners, and any agents you register yourself
- **Agents map** - Visualize agent relationships and dependencies across your organization
- **Agent analytics** - Monitor agent performance and usage patterns
- **Role-specific oversight** - Enable tailored management based on organizational roles

## Manage agents by extending the systems you use for employees

Observe, secure, and govern every agent across your organization with familiar systems, tailored to the unique needs of agentic AI:

- **Microsoft Defender** - Extend comprehensive security posture and advanced threat protection to agents
- **Microsoft Entra** - Protect agent identities and secure access to apps, resources, internet, and other agents
- **Microsoft Purview** - Manage, protect, and govern data that agents use and create across your organization
- **Microsoft 365 admin center** - Centralized hub to manage users, agents, and settings securely across your Microsoft 365 environment
